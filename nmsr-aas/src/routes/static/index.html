NickAc's Minecraft Skin Renderer

A service written in Rust to render Minecraft skins with true perspective.
This works by using pre-rendered UV maps that are then merged together into the output image.

API endpoints

Listed here are all the API endpoints that are available.

GET /full/{uuid} - Render a player's full body with their current skin

Render a player's current Minecraft skin onto a PNG image.

You can also specify the ?alex query parameter to use the Alex model.

Examples:

  • Full body render of NickAc
  • Full body render of Koide
  • Full body render of BmaxXSux

GET /skin/{uuid} - Fetch a player's current skin

Get the player's current Minecraft skin returned as a PNG image.

Examples:

  • Skin of NickAc
  • Skin of Koide
  • Skin of BmaxXSux
